# Tableu-HR-dashboard

This Tableau dashboard provides a complete overview of employee workforce data and attrition trends. It helps HR teams analyze key metrics such as Employee Count (1470), Attrition Count (237), Attrition Rate (16.12%), Active Employees (1233), and Average Age (37).

The dashboard includes visualizations for department-wise attrition, employee age distribution, job satisfaction ratings, education field-wise attrition, and gender-based attrition analysis. It also compares attrition rates across different age groups for male and female employees.
This dashboard is useful for identifying turnover patterns, improving employee retention strategies, and supporting data-driven HR decision-making.


Tools & Technologies Used:-
Tableau – Data visualization and dashboard creation
HR Analytics Dataset – Employee and attrition data
Calculated Fields – For KPI metrics like attrition rate and active employees
Charts Used – Pie Chart, Bar Chart, Histogram, Heatmap, Donut Chart, KPI Cards
Filters – Interactive filtering for better analysis
